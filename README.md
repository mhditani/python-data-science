# 📊 Python Data Analysis & Visualization Project

## 📌 Overview

This project is a hands-on exploration of **NumPy, Pandas, and Matplotlib** using Jupyter Notebook. It demonstrates fundamental concepts in data manipulation, analysis, and visualization through practical examples.

The project covers working with arrays, dataframes, statistical analysis, and multiple types of plots.

---

## 🚀 Features

* NumPy array creation and manipulation (1D, 2D, 3D)
* Random data generation and mathematical operations
* Data analysis using Pandas (DataFrames & Series)
* Data cleaning and exploration (`head`, `tail`, `info`, `describe`)
* Grouping and aggregation (`groupby`, `crosstab`)
* Data filtering and indexing (`loc`, `iloc`)
* Exporting data to CSV
* Data visualization with Matplotlib:

  * Line plots
  * Scatter plots
  * Bar charts
  * Histograms
* Saving plots as image files

